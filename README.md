# Simple Inventory App

## Description
This is a **basic inventory management app** written in Python.  
It is designed for small inventories and is easy to use.  
The app allows users to **add products, view them, and calculate the total value** of all products in stock.

It is a simplified version of a full inventory system, ideal for **learning Python** or small personal projects.

---

## Features
- **Add products**  
  You can enter a product name, price, and quantity.  
  The app validates that the **price is a positive number** and the **quantity is a positive integer**.

- **Show products**  
  Displays all products in the inventory in a **clear format** with name, price, and quantity.  
  If the inventory is empty, it shows a message saying there is nothing to display.

- **Calculate total value**  
  The app calculates the **total value of the inventory** by multiplying price × quantity for each product and adding them all.  
  If there are no products, it shows a message saying there is nothing to calculate.

---

- Python 3.x  
- No additional libraries required
